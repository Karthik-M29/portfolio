Satellite Swarm Communication System

2026 | Arduino IDE

A satellite swarm communication system is a network of multiple small satellites that work together to transmit data using a multi-hop communication method, where each satellite acts as both a transmitter and receiver. In this project, components like Arduino UNO, nRF24L01 RF modules, AMS1117 regulator, capacitor, LED, and an LCD (for ground station) are used to simulate real satellite behavior. The working principle is that data is sent from one node and forwarded through intermediate nodes until it reaches the ground station, with the last active node responsible for final transmission. If any node fails, the system automatically reroutes the data through another available node, ensuring fault-tolerant and continuous communication. The main function of this system is to provide reliable data transfer without interruption, similar to modern satellite networks. The unique aspect of this project is its self-recovery capability, dynamic routing, and decentralized operation, unlike traditional single-satellite systems. In the future, this system can be enhanced with long-range communication, IoT sensors, and intelligent routing algorithms, making it closer to real-world satellite swarm applications used by organizations like NASA and ESA.

Day 02 - OR Gate

07 Apr 2026 | EDA Playground

Explanation: The OR gate outputs logic 1 when at least one input is HIGH (1). The Verilog design was implemented and simulated in EDA Playground using a testbench that applies all possible input combinations (00, 01, 10, 11) over time. Waveform: From the waveform, the output remains LOW (0) only when both inputs are 0. For all other cases (01, 10, 11), the output transitions to HIGH (1). The timing of input changes and corresponding output confirms correct logical behavior and proper propagation in the simulation.
